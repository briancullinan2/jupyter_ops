Here is a summary of the code in two sentences:

This code imports dependencies and defines Selenium cell tasks to log in and unfollow everyone on Facebook using the runSeleniumCell function, which is executed asynchronously. The tasks are then executed sequentially using a promise chain, with the results sent to the $$.sendResult function and any errors sent to the $$.sendError function.

var importer = require('../Core');
var runSeleniumCell = importer.import('selenium cell');
// Unfollow everyone on facebook?

// https://www.facebook.com/me/following
var loginFacebook, unfollowFacebook;
$$.async();
runSeleniumCell([
    'log in facebook',
    'unfollow everyone facebook'
])
    .then(r => {
        loginFacebook = r[0];
        unfollowFacebook = r[1];
        return loginFacebook();
    })
    .then(() => unfollowFacebook())
    .then(r => $$.sendResult(r))
    .catch(e => $$.sendError(e));
