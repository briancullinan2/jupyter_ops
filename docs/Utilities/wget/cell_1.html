wget Command Breakdown

Description

This is a wget command used for downloading a website, specifically excluding certain directories.

Options

  • --mirror: Recursively retrieve the entire site, including files and directories.
  • --convert-links: Convert links to point to local files, making them usable offline.
  • -nc: Do not continue if a single file fails to download (no continuation).
  • -r: Turn on recursive retrieving (similar to --mirror).

Excluded Directories

  • /forums
  • /files
  • /images
  • /gallery
  • /nc
  • /diane

URL

http://www.truecombatelite.com/mapdepot/
