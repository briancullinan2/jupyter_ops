Highlight.js styles and script, along with additional language scripts, are imported and initialized. The highlight.js plugin and all code in the document are then added and highlighted.

Run example

npm run import -- "highlight js"

Code Breakdown

Importing Highlight.js Styles and Script

  • Two CSS styles are imported:
    • default.min.css: The default highlight.js style.
    • github-dark.css: A dark-themed style inspired by GitHub.
  • The highlight.js script is imported.

Loading Additional Languages

  • A script is loaded dynamically for a specific language (${LANGUAGE}).
  • An example script is loaded for the Bash language.
  • A plugin script is loaded (../../mergehtml.js).

Initialization Script

  • The addPlugin function of highlight.js is called to add a plugin named mergeHTMLPlugin.
  • The highlightAll function is called to highlight all code in the document.
