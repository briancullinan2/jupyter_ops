askOtherLLMs Function

Description

Asynchronous function to ask other Large Language Models (LLMs) a question.

Parameters

  • promptModel: The model to use for generating the prompt.
  • session: The session object for the LLM.
  • prompt: The question to ask the LLM.

Returns

An asynchronous value, currently undefined.
