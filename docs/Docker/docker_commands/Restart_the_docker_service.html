This code checks for a running Docker container by name and, if found, stops and removes it.

if docker ps | grep "{name}" then; \
    docker stop {name}; \
    docker rm {name}; \
fi

This code snippet checks if a Docker container with a specific name ("{name}") is running and, if so, stops and removes it.

Here's a breakdown:

  1. docker ps | grep "{name}":

    • Executes the docker ps command to list running containers.
    • Pipes the output to grep to search for a line containing the specified container name ("{name}").

  2. then;:

    • If grep finds a match (meaning the container is running), the code block following the then statement is executed.

  3. docker stop {name};:

    • Sends a STOP signal to the container with the specified name.

  4. docker rm {name};:

    • Removes the stopped container with the specified name.

  5. fi:

    • Marks the end of the if statement.

In essence, this code provides a concise way to stop and remove a Docker container if it is currently running.
