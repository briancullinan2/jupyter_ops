The createdataframe function takes a directory path as input and returns two lists: image_paths and labels, containing the paths to images and their corresponding labels, respectively. It traverses the directory structure to extract this information.

def createdataframe(dir):
    image_paths = []
    labels = []
    for label in os.listdir(dir):
        for imagename in os.listdir(os.path.join(dir,label)):
            image_paths.append(os.path.join(dir,label,imagename))
            labels.append(label)
        print(label, "completed")
    return image_paths,labels

Function: createdataframe

Parameters

  • dir: The path to the directory containing images and labels

Returns

  • image_paths: A list of paths to the images
  • labels: A list of corresponding labels

Description

This function traverses a directory structure, extracting paths to images and their corresponding labels. It returns two lists: image_paths containing the paths to the images and labels containing the corresponding labels.

Implementation

  • It iterates over the subdirectories (labels) in the specified directory.
  • For each subdirectory, it iterates over the image files and appends the full path to the image and the corresponding label to the image_paths and labels lists, respectively.
  • It prints a message indicating the completion of each label.
  • Finally, it returns the image_paths and labels lists.
