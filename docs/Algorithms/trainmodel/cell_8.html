The extract_features function extracts features from a list of images and returns a 4D numpy array containing the features. It assumes images are loaded as grayscale arrays and does not perform any preprocessing or feature extraction.

def extract_features(images):
    features = []
    for image in tqdm(images):
        img = load_img(image,grayscale =  True )
        img = np.array(img)
        features.append(img)
    features = np.array(features)
    features = features.reshape(len(features),48,48,1)
    return features

Function: extract_features

Purpose

Extracts features from a list of images.

Parameters

  • images: A list of image file paths or numpy arrays.

Returns

A 4D numpy array of shape (n_samples, 48, 48, 1) containing the features of the input images.

Notes

  • Uses tqdm for progress tracking (not shown in code snippet).
  • Assumes load_img and np.array are functions from the Keras or OpenCV libraries.
  • Reshapes the output to match the shape expected by deep learning models.
  • Does not perform any preprocessing or feature extraction; simply loads images as grayscale arrays.
