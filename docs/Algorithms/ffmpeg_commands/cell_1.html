Here's a 2-sentence summary of the FFmpeg command:

The FFmpeg command is used to stream video and audio from an RTMP source to YouTube Live, using the H.264 codec for video and the AAC codec for audio. The command sets various settings for video and audio encoding, including bitrate, frame rate, and sampling rate, and outputs the stream in FLV format at a specified URL.


ffmpeg -i http://192.168.4.22:8080/video -i http://192.168.4.22:8080/audio.wav -c:v libx264 -preset ultrafast -b:v 4500k -maxrate 4500k -bufsize 9000k -g 50 -c:a aac -b:a 128k -ar 44100 -f flv rtmp://a.rtmp.youtube.com/live2/dvqv-7eyf-72d5-vaw7-dvsb

Command Breakdown

FFmpeg Command

ffmpeg -i http://192.168.4.22:8080/video -i http://192.168.4.22:8080/audio.wav -c:v libx264 -preset ultrafast -b:v 4500k -maxrate 4500k -bufsize 9000k -g 50 -c:a aac -b:a 128k -ar 44100 -f flv rtmp://a.rtmp.youtube.com/live2/dvqv-7eyf-72d5-vaw7-dvsb

Options and Parameters

Input Settings

  • http://192.168.4.22:8080/video: Video input URL
  • http://192.168.4.22:8080/audio.wav: Audio input URL

Video Settings

  • -c:v libx264: Use H.264 codec for video
  • -preset ultrafast: Use ultrafast preset for H.264 encoding
  • -b:v 4500k: Set video bitrate to 4500 kbps
  • -maxrate 4500k: Set maximum video bitrate to 4500 kbps
  • -bufsize 9000k: Set video buffer size to 9000 kbps
  • -g 50: Set GOP size to 50 frames

Audio Settings

  • -c:a aac: Use AAC codec for audio
  • -b:a 128k: Set audio bitrate to 128 kbps
  • -ar 44100: Set audio sampling rate to 44.1 kHz

Output Settings

  • -f flv: Set output format to FLV
  • rtmp://a.rtmp.youtube.com/live2/dvqv-7eyf-72d5-vaw7-dvsb: Output URL for RTMP streaming
