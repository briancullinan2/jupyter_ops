The Docker container is run with interactive shell access, mapping port 8888 and mounting a directory to share files between the host machine and the container.

Breakdown of the Code

Section 1: Clone a GitHub Repository

git clone https://github.com/megamindbrian/jupytangular2
  • This line clones the repository located at https://github.com/megamindbrian/jupytangular2 using the git clone command.

Section 2: Build a Docker Image

docker build -t jupytangular.
  • This line builds a Docker image with the tag jupytangular from the current directory (.).

Section 3: Run a Docker Container

docker run --name jupytangular -it -p 8888:8888 -v ~/jupytangular2/notebooks:/home/jovyan/notebooks jupytangular
  • This line runs a Docker container with the name jupytangular from the jupytangular image.
  • The -it flag allows for interactive shell access.
  • The -p 8888:8888 flag maps port 8888 on the host machine to port 8888 in the container.
  • The -v flag mounts the directory ~/jupytangular2/notebooks on the host machine to /home/jovyan/notebooks in the container.
  • The last part of the command, jupytangular, is the name of the Docker image to run.

Section 4: Run Another Docker Container (Same as Section 3)

docker run --name jupytangular -it -p 8888:8888 -v 'C:\Users\brian.cullinan\Documents\jupytangular2\notebooks':/home/jovyan/notebooks jupytangular
  • This line is identical to the previous command, but with a different path for the mounted directory. The path is now in Windows format (C:\Users\brian.cullinan\Documents\jupytangular2\notebooks).
