The $TS.screen() function takes a URL or domain (act.com) and an object with properties such as zoom level, width, and crop coordinates to capture a screenshot of a webpage. This function can be customized to capture a specific area of the webpage based on the provided parameters.

Cell 1

$TS.screen('act.com', {zoom: .5, width: 680, 'crop-y': 400, 'crop-h': 400});

Function Call: $TS.screen()

  • Parameters:
    • act.com: URL or domain of the website to be screened.
    • Object with the following properties:
      • zoom: Zoom level of the webpage (0-1 range).
      • width: Width of the screenshot (in pixels).
      • crop-y and crop-h: Cropped area of the screenshot (y and height coordinates).
