class ListNode(object):
    def __init__(self, x):
        self.val = x
        self.next = None
        
    def __str__(self): # for your debugging purposes
        return str(self.val) + '->' + str(self.next)
    
    def __eq__(self, other): # for asserts to work
            return (isinstance(other, self.__class__) 
                and self.__dict__ == other.__dict__)
        
def reverse_list(head):
    """Iterative solution."""
    prev = None
    while head:
        nxt = head.next
        head.next = prev
        prev = head
        head = nxt
    return prev

Parameters

Return Value

The new head of the reversed linked list.

Algorithm

  1. Initialize three pointers: prev, head, and nxt.
  2. Traverse the linked list:
    • Set nxt to the next node in the list (head.next).
    • Set head.next to prev, effectively reversing the link between the current node and its next node.
    • Move prev and head one step forward.
  3. Repeat step 2 until the end of the list is reached (head becomes None).
  4. Return the new head of the reversed linked list (prev).
