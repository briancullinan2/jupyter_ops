The regexToArray function takes a regular expression object and a string, and returns an array of arrays or numbers containing the index and length of each match. It can also return a single match at a specified index.

function regexToArray(ex, str, i = 0) {
    var co = []
    var m
    while ((m = ex.exec(str))) {
        co.push(i === -1 ? [m.index, m[0].length] : i === false ? m : m[i])
    }
    return co
}

module.exports.regexToArray = regexToArray;

Function: regexToArray

Parameters

  • ex: A regular expression object
  • str: The string to search
  • i: The index of each match (default: 0)

Return Value

  • An array of arrays or numbers

Description

The regexToArray function takes a regular expression object ex and a string str, and returns an array of arrays or numbers. The function uses the regular expression to find matches in the string and returns the index and length of each match, or a single match at the specified index i.

Example Use Case

const regexToArray = require('./regexToArray');

const regex = /ab/g;
const str = 'ababab';
const result = regexToArray(regex, str);
console.log(result); // Output: [[0, 2], [3, 2], [6, 2]]
