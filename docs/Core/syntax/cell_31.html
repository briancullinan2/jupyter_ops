  • var CALLS: declares a variable named CALLS.
  • =: assigns a value to the variable.
  • ``: defines a string literal, which is an "escape" string containing a JavaScript comment //CallExpression.
    • The backticks `` indicate a template literal.
    • The double forward slashes // start a line comment in JavaScript, which is discarded by the interpreter.
    • The string 'CallExpression' is the actual value assigned to the CALLS variable.
