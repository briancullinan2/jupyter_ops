This code imports modules, performs file system operations, and processes Jupyter notebooks by reading, updating, and writing them back to the file system. It searches for notebooks in a specified project path and its subdirectories.

var fs = require('fs');
var importer = require('../Core')
var {glob} = importer.import("glob files")
fs.readdirSync('.');
var PROJECT_PATH = '/Users/briancullinan/Google Drive/Colab Notebooks'
glob.sync('**/*.ipynb', {cwd: PROJECT_PATH}).forEach(f => {
    const nb = JSON.parse(fs.readFileSync(path.join(PROJECT_PATH, f)).toString());
    nb.nbformat_minor = 0;
    /*
    nb.metadata = {
        colab: {

          "provenance": []
        },
        kernelspec: {
          "name": "javascript",
          "display_name": "Javascript"
        }
    };
    */
    fs.writeFileSync(path.join(PROJECT_PATH, f), JSON.stringify(nb, null, 2));
});

Code Breakdown

Importing Modules

  • var fs = require('fs');: Import the Node.js file system module.
  • var importer = require('../Core'): Import a custom module named Core from a parent directory.
  • var {glob} = importer.import('glob files'): Import the glob module from the Core module, specifically loading files related to globbing patterns.

File System Operations

  • fs.readdirSync('.');: Read the contents of the current working directory synchronously.
  • var PROJECT_PATH = '/Users/briancullinan/Google Drive/Colab Notebooks': Define a constant for a specific project path.

Finding and Processing Jupyter Notebooks

  • glob.sync('**/*.ipynb', {cwd: PROJECT_PATH}): Search for Jupyter notebooks (*.ipynb) in the project path directory and its subdirectories synchronously.
  • The result is then processed in a forEach loop:
    • const nb = JSON.parse(fs.readFileSync(path.join(PROJECT_PATH, f)).toString());: Read each notebook file as a JSON object.
    • nb.nbformat_minor = 0;: Update the notebook's minor format version.
    • fs.writeFileSync(path.join(PROJECT_PATH, f), JSON.stringify(nb, null, 2));: Write the updated notebook back to the file system with indentation.
