/**
 * Changes the root path of a file.

 * @returns {string} The file path with the new root.

function changeRoot(file, root, output) {

Code Breakdown

Overview

This code exports two functions, chext and chroot, which manipulate file paths.

Functions

chext(file, ext)

  • Replaces the file extension of file with ext.
  • Uses a regular expression to match the file extension of file and replace it with ext.
  • Returns the modified file path.

chroot(file, root, output)

  • Checks if file starts with root (case-sensitive).
  • If file does not start with root, throws an error.
  • Otherwise, returns a new file path by joining output and the part of file after root.
  • Uses the path module to manipulate file paths.

Module Exports

The code exports an object with the two functions, chext and chroot.
