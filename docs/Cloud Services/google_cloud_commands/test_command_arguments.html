This code snippet simply prints the values of three command-line arguments to the console.

Here's a breakdown:

  1. echo "$1": Prints the value of the first command-line argument ($1).
  2. echo "$2": Prints the value of the second command-line argument.

echo "$3": Prints the value of the third command-line argument.

Explanation:

  • echo: This is a command in Unix-like operating systems that prints its arguments to the standard output (usually the console).
  • "$1", "$2", "$3": These are variables that represent the command-line arguments passed to the script.
    • $1 refers to the first argument.
    • $2 refers to the second argument.
    • $3 refers to the third argument.
    • The double quotes (") around the variables are important because they ensure that any spaces or special characters within the arguments are treated as part of the argument itself.

How it works:
